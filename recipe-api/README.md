# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command

## Prerequisites

npm install npm@latest -g

### Stack used
* Node.js
* TypeScript
* Express.js
* ESLint
* Json Web Token
* Bcrypt
* Mysql
* Apollo Server
